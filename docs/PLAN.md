## 2. 데이터 소스 판정

- **결론: smok95/lotto GitHub 미러 사용** (1회차~최신, all.json, 매주 자동 갱신).
  node에서 정상 수신 확인. `scripts/fetch-draws.mjs` → `data/draws.json`.
- **동행복권 공식 엔드포인트는 사용 불가**: WAF가 비브라우저 요청을 홈/errorPage로
  리다이렉트. 세션 쿠키·브라우저 헤더·실제 브라우저 엔진 모두 차단 확인.
- 당첨번호는 사실(fact)이라 재게시 제약 없음. 저빈도(주 1회) 정적 수집.
- 공식 확인 안내(dhlottery.co.kr) + 정확성 미보증 고지 유지.

## 3. 사이트 구조 (구현 완료)

```
lotto.lifebanjang.com
├─ /            홈 — 최신 회차 + 도구 카드 + 최근 회차
├─ /numbers     회차별 당첨번호 + 1~5등 당첨금 표 (최근 100회)
├─ /stats       번호별 출현 통계·미출현 기간·홀짝/고저 분포
├─ /generator   번호 생성기 (랜덤/통계가중, 고정수·제외수, 다게임)
├─ /guide       당첨 확률·세금·수령방법 + FAQ(JSON-LD)
└─ /about /privacy /terms
```

SEO 타깃: `로또 당첨번호`, `로또 번호 생성`, `로또 통계`, `이번주 로또`, `로또 확률`.
